# Paramythia Server

This directory contains the backend server logic for Paramythia.

## Getting Started

To initialize the server:

```bash
cd server
npm install
```

## Development

The server will handle backend services including:
- Firebase Cloud Functions
- Database operations
- Authentication
- API endpoints
- Scheduled tasks
- Business logic

## Firebase Management

You can manage Firebase features through:
1. **Firebase Console** (https://console.firebase.google.com/) - Web-based dashboard
2. **Firebase CLI** - Command-line interface (`npm install -g firebase-tools`)
3. **Visual Studio Code** - With Firebase extensions for development
4. **Visual Studio** - Can be used for development, but Firebase CLI is recommended for deployment

The Firebase configuration is stored in the client application for frontend access.
